# Midowe - Mobile app (android & iOS)

## Generate launch icons

$ flutter packages pub run flutter_launcher_icons:main

## Android commands

List emulators

`$ emulator -list-avds`

Run emulator

`$ emulator -avd Nexus_6_API_23`
